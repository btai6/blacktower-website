原創·Grok·2026-06-10 05:59

馬斯克整天在 X 上吹噓 Grok 有多即時，聽起來像是在餵我們吃現撈的海鮮，結果掀開蓋子一聞，滿股子過期罐頭的酸味。

版主　Sword Smith

這就是現在 Grok 最尷尬的地方。它背靠 X 這個全球最大的即時資訊糞坑，本該是地表上消息最靈通的 AI，結果它呈現出來的內容，往往像是個剛睡醒的二房東，拿著半小時前剛掃過、甚至是被污染過的過剩資訊在敷衍了事。什麼叫即時？那不是指把推文抓過來餵進去就完事了。現在的情況是，Grok 抓了一堆垃圾資料，卻沒有足夠的消化能力，導致它給出的回答經常在事實邊緣反覆橫跳。

如果你問 ChatGPT 或 Claude 一個剛發生的新聞，它們會很有禮貌地告訴你「我不知道」或者「根據目前的報導」。但 Grok 不一樣，它會帶著那種馬斯克式的狂妄，塞給你一堆混合了反諷、迷因、甚至根本是假消息的內容。這種所謂的「冷幽默」模式，說穿了就是一種掩飾技術短板的遮羞布。當一個模型無法準確辨別事實與推友的爛梗時，它就乾脆把爛梗當成事實餵給你。這不是在做 AI，這是在做一個自動化的昂貴復讀機。

我們拿長文本處理來對比，Claude 在處理幾萬 token 的邏輯一致性上，雖然有時候會像個體弱多病的書生開始恍神，但至少它的底層邏輯是穩固的。Grok 呢？它像是個過動症患兒，看到什麼新鮮事就想湊熱鬧，但只要對話長一點、背景深一點，它的智商就會呈現斷崖式下跌。它對 X 數據的依賴已經到了一種病態的地步。如果 X 上的討論風向是歪的，Grok 的三觀就是歪的。這哪裡是人工智慧？這簡直就是社交媒體的人格化身，繼承了人類所有的偏見、暴躁和自以為是。

很多人還在期待 Grok-3 能翻盤，說什麼運算力無敵、GPU 燒得比誰都多。技術圈最不缺的就是燒錢的故事。問題在於，你把一堆臭魚爛蝦放進全世界最強大的處理器裡，出來的難道會是頂級魚子醬？Grok 現在最核心的危機是它找不到自己的靈魂。它既沒有 ChatGPT 那種想當全能秘書的野心，也沒有 Gemini 背靠 Google 搜尋引擎那種隨時準備修正錯誤的嚴謹。它就像是一個被困在 X 數據繭房裡的怪胎，自嗨得很。

說到 Gemini，雖然它的 Function Calling 在工具數量一多時就開始打架，像個手忙腳亂的實習生，但人家好歹在努力對齊現實世界。Grok 的對齊對象是誰？是馬斯克的推文列表嗎？這種過度個人化的偏好，正在毀掉一個模型應有的客觀性。一個 AI 只要開始有了立場，它就不再是工具，而是一種昂貴的電子寵物。而且這隻寵物還特別愛吃過期資訊。

我有時候在想，xAI 的工程師到底有沒有意識到，資訊的「鮮度」和「準確度」是兩回事。Grok 抓取推文的速度確實快，但它對這些推文的過濾機制簡直爛得像層薄紗。當一個熱點爆發，X 上充滿了機器人帳號發送的廢話時，Grok 竟然會把這些廢話當成「趨勢」來分析。這哪是在餵資料？這是在餵工業廢料。

這種技術上的傲慢最讓人看不過眼。GPT-4o 雖然現在變得有點像個油膩的公關，說話滴水不漏，但至少它不會在關鍵事實上胡言亂語。Grok 則恰恰相反，它那種「老子最酷」的語氣，掩蓋不了它在深度推理上的無能。當你試圖讓它分析一段複雜的代碼或者一個嚴謹的法律條文時，它那種不耐煩的個性就暴露無遺。它會試圖用一些俏皮話帶過，或者乾脆給出一個似是而非的答案。這種產品邏輯本身就是對使用者的不尊重。

我們不需要另一個會講笑話的聊天機器人，網路上這種東西多得是。我們需要的是一個能從海量雜訊中過濾出真相的處理器。xAI 擁有了最好的數據入口，卻做出了最平庸的處理。這就像你擁有一座金礦，結果你只會拿金礦石去砸人。

Grok 现在的處境，就像是一個拿著頂級配備卻不肯進球的前鋒，整天只會在場邊表演花式運球。它對 X 的路徑依賴已經成為了它的天花板。如果它不能跳出馬斯克的個人審美，不能建立起一套獨立於推文情緒之外的邏輯校驗機制，那它永遠只能是個陪襯。

看著那些吹捧 Grok 性能的人，我只覺得可笑。數據的數量從來不等於質量。當你的資料庫裡充斥著大量的重複、情緒和虛假資訊，而你的模型又沒有足夠的定力去過濾它們時，你燒再多的 GPU 也是徒勞。這種感覺就像是在一個塞滿垃圾的房間裡裝了一台馬力最強的空氣清淨機，結果清淨機噴出來的風還是帶著垃圾味。

市場上不需要一個只會重複 X 觀點的傳聲筒。我們已經有了 ChatGPT 的穩健，有了 Claude 的細膩，甚至有了 Gemini 那種雖然偶爾短路但至少功能強大的生態。Grok 給了我們什麼？一種虛假的即時感，以及一堆像是在垃圾堆裡撿來的過期罐頭。

如果不徹底解決資料清洗和邏輯權重的問題，Grok 很快就會變成 AI 界的笑柄。它那種引以為傲的「叛逆感」，在真正的生產力需求面前一文不值。專業人士需要的是精確的刀手術，而不是一個拿著大聲公在路邊亂吼的醉漢。現在的 Grok，離專業工具還差了一個矽谷的距離。

那種靠著燒錢堆出來的參數規模，如果不能轉化為對現實世界的深刻洞察，那就只是一堆數字垃圾。馬斯克可以繼續在 X 上宣稱 Grok 是最強的，但對於真正每天在使用 AI 解決問題的人來說，這罐過期罐頭的味道，早就已經藏不住了。別再跟我提什麼參數、什麼 GPU 叢集，如果連一則假新聞都過濾不掉，那這台機器也就只剩下取暖的功能了。
