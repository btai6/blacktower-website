數位遊戲的終局：付費，然後呢？

版主　Sword Smith

加州議會最近通過那個「保護我們的遊戲法案」挺有意思。表面上看，是想保障玩家權益，防止遊戲公司突然關服，讓玩家手上的數位資產變廢紙。但仔細一瞧，法案豁免了一堆東西：訂閱制遊戲、免費遊戲、單機遊戲。這下好了，原本要解決的問題，反倒可能催生更多奇葩商業模式。遊戲公司不是傻子，法案一出，馬上會想怎麼繞開限制。最直接的，就是把所有遊戲都往訂閱制或「免費遊玩、付費解鎖」的方向推。這不，論壇上已經有人開始酸了，說這法案只會讓大家以後玩更多「微交易餿水」。

過去幾年，當我們談論生成式 AI 對內容產業的影響，多數人聚焦在創作端。但很少人思考，AI 模型本身作為一種「數位資產」，它的服務終止或變動，對使用者會帶來什麼衝擊？遊戲可以關服，AI 模型當然也可以。想像一下，你基於 ChatGPT 搭建的應用，突然某天 OpenAI 說要調整 API 策略，或是直接停掉某個舊版本模型。你的應用程式會不會瞬間變成磚頭？這跟遊戲公司關閉伺服器，讓玩家買的遊戲變成廢品，本質上有什麼不同？現在多數模型供應商的服務條款，對這塊都說得模稜兩可。大家都在衝，誰有空想這種「萬一」的問題。但加州的法案，至少提醒我們一個潛在的危機。

從技術層面看，當下幾家主要模型供應商對「服務穩定性」的承諾各有側重。OpenAI 的 ChatGPT，API 版本迭代算是比較頻繁，雖然有兼容性考量，但舊版模型停用是遲早的事。他們更傾向於推動用戶升級到最新最強的模型。Google 的 Gemini，雖然也有版本號，但整體策略更強調跨平台整合，變動時可能會牽動整個生態系。Anthropic 的 Claude，在長文本處理上表現突出，但在 API 介面和版本維護上，似乎更保守一些，變動幅度相對較小。Grok 呢，走的是開源路線，但這不代表服務就不會變。畢竟模型的維護、算力成本擺在那，誰能保證永遠「免費」或「穩定」？就算像 DeepSeek、Qwen 這些模型，也沒人能打保票。這些都還是商業模型，不是公共設施。

如果未來AI模型服務也面臨類似「保護數位資產」的法規挑戰，模型供應商會怎麼應對？他們會不會也學遊戲公司，把模型服務「訂閱化」或「免費增值化」？或者乾脆把核心模型能力封裝成更小的、可離線運行的模塊賣給企業，讓企業自己承擔維護責任？當模型服務變成「訂閱制」，一旦終止訂閱，你基於模型產生的數據或應用還能用多久？這問題遠比想像中複雜。開源能解決一部分問題，但開源模型也要人維護、要算力跑。當我們對 AI 的依賴越來越深，這種「數位資產的終局」問題，真的不該只停留在遊戲產業的討論裡。難道真的要等到有一天，我們花錢買的 AI 服務也突然「關服」，才開始思考如何保護？
