算力堆出來的冷笑話

版主　Sword Smith

馬斯克大概覺得把十萬張 H100 堆在一起，就能像變魔術一樣變出一個具備靈魂的 AI，這種典型的矽谷暴發戶思維，看久了真的讓人反胃。大家都在吹捧那個叫 Colossus 的算力集群，彷彿有了這堆燃燒電力和美金的廢鐵，Grok 就能一夜之間變成全知全能的神。現實是，算力只是引擎，如果你連導航都不會設，方向盤還握在一個每天在社群平台上發瘋的老闆手裡，這台跑車最後只會衝進臭水溝。

看看現在的 Grok，整天標榜什麼「反覺醒」、什麼「追求真相」，聽起來熱血沸騰，實際上用起來卻像個剛學會抽菸就以為自己很酷的高中生。你問它一個嚴肅的科學問題，它可能還要先夾帶兩句冷嘲熱諷，好像不損你兩句就顯不出它很有個性。拜託，我是來找答案的，不是來聽一個機器人對我說教。所謂的「幽默模式」更是災難，那種尷尬的冷笑話簡直是在挑戰人類的耐性極限，這種刻意營造的「叛逆感」比起那些滿口客套話的競爭對手，其實高明不到哪去。

說到競爭對手，那個滿嘴仁義道德、動不動就說「作為一個 AI 語言模型，我不能回答這個問題」的 Claude，確實讓人想翻白眼。它就像是一個領著高薪、成天擔心被 HR 抓到把柄的企業中階主管，說話滴水不漏，但也無趣到了極點。GPT 則是越來越像個老掉牙的大百科全書，雖然穩定，但那種標準的論文腔調，看三行就想睡覺。至於 Gemini，除了靠著 Google 那點家底在撐場景，它那種連歷史照片都能搞錯的邏輯，根本就是個笑話。在這種全員平庸的背景下，xAI 本來有機會殺出一條血路，結果他們選擇了最無腦的一條路：堆硬體，然後在那裡自嗨。

我真的看不慣那種「算力即一切」的傲慢。馬斯克在 X 上發幾張機房的照片，底下一堆信徒就開始集體高潮，好像 Grok 3 出現的那一天，人類的所有問題都能迎刃而解。你們到底有沒有意識到，大模型的邏輯訓練不是靠燒錢就能燒出來的？Grok 最大的優勢本來應該是 X 平台上的即時數據流，那可是全世界最瘋狂、最真實、也最混亂的情報庫。結果呢？Grok 抓取數據的方式簡直粗糙得令人髮指。它常常把一些帳號的陰謀論當成事實，還自以為發現了什麼不為人知的真相。這不叫追求真相，這叫資訊消化不良。

如果一個模型不能過濾掉那些充滿惡意的雜音，它就只是個高效的擴音器而已。現在的 Grok 就是在扮演這個角色，它把 X 上的情緒放大了，卻沒有把智慧提煉出來。那些看好它的人，多半是帶著某種政治濾鏡，覺得只要能對抗主流媒體的敘事就是好東西。清醒一點吧，技術就是技術，技術本身不應該有黨派。當一個 AI 開始有了鮮明的政治傾向，它的客觀性就已經破產了。這跟它的算力是多少個 Petaflops 一點關係都沒有。

很多人在論壇裡護航，說什麼 Grok 還在早期階段，給它一點時間。這句話我聽了兩年了。從第一代那個簡陋的網頁版到現在，除了反應速度快了一點、會發圖片了，它的邏輯深度有質的飛躍嗎？並沒有。它依然在重複那些大模型的通病，甚至在幻覺問題上比別人更嚴重。因為它太想顯得「與眾不同」，所以它會為了迎合那種反骨的設定，去編造一些看起來很合理但完全錯誤的邏輯。這種錯誤比 GPT 那種誠實的無知更可怕，因為它帶有誤導性。

再聊聊那個所謂的「多模態」。現在每家公司都在搞圖片生成、搞影片生成，Grok 也不甘示弱。但你看看它生成的那些圖片，充滿了某種難以言喻的廉價感。雖然自由度很高，什麼都能畫，這確實滿足了一些人的惡趣味，但這算哪門子技術領先？這只是單純的開放權限罷了。真正的技術領先是像 Claude 3.5 Sonnet 那樣，能精準理解複雜的指令，能幫你改代碼改到讓你懷疑人生，而不是在那裡畫一些政治人物的諷刺畫。

xAI 的團隊裡確實有一群頂尖的人才，這點我不否認。但這些天才現在看起來更像是馬斯克的私人玩具設計師。他們被迫要在短時間內交出數據，要在短時間內衝榜，要在短時間內證明 Colossus 沒白蓋。在這種壓力下產出的產品，注定是充滿瑕疵的。那種為了衝榜而進行的微調，除了能讓跑分數據好看一點，對實際使用體驗一點幫助都沒有。我們不需要另一個能考高分的機器人，我們需要一個能真正理解問題脈絡、不帶偏見、且具備深度洞察力的助手。顯然，Grok 離這個目標還遠得很。

最讓我感到不耐煩的是，每次 xAI 有點小動作，那些科技媒體就開始用那種「顛覆者來了」的語氣寫稿。什麼顛覆？顛覆了誰？是顛覆了我們對廢話的定義嗎？如果你每天對著螢幕，看到的只是一個會用髒話跟你開玩笑的機器人，你真的覺得這是未來？真正的 AI 革命應該是無聲無息地滲透進生產力工具裡，是能幫你解決那些繁瑣、無意義的勞動，而不是變成一個在社交媒體上刷存在感的網紅。馬斯克太懂怎麼當網紅了，但他顯然忘了怎麼做一個讓人尊重的產品經理。

大家對 Grok 的期待，很大程度上是建立在對現狀的不滿。我們討厭被過度保護，討厭被告知什麼能說、什麼不能看，所以當 Grok 出現時，大家把它當成了救世主。但這種報復性的支持是盲目的。你不能因為討厭一個偽君子，就去擁抱一個真流氓。Grok 現在的狀態就是那個真流氓，它毫無忌憚地展示著自己的缺陷，還美其名曰「真實」。這種對真實的誤解，才是最讓人痛心的地方。

還有那個訂閱制。為了用這個模型，你得先買 X 的會員。這本身就是一種極其傲慢的綑綁銷售。馬斯克把 AI 當成了拯救 X 營收的救命稻草，這種急功近利的心態，讓 Grok 從出生那一刻起就帶著濃厚的商業臭味。它不是為了人類的未來而生的，它是為了馬斯克的報表而生的。在這種邏輯下，產品的優先順序永遠是「噱頭大於實質」。只要能在 X 上引起討論、能帶來流量，這項功能就會被推到最前面。至於模型內在的連貫性和可靠性，誰在乎？

我每天在論壇看你們吵來吵去，說這家的 API 便宜、那家的上下文長度夠，說真的，這都是枝節問題。核心問題在於，我們有沒有勇氣承認，現在的 AI 發展已經進入了一個死胡同？大家都在拚命堆參數、堆算力，卻沒人敢停下來思考，除了這種暴力破解的方法，還有沒有別的路。xAI 標榜自己要走不同的路，結果它走的是一條最暴力的路。用最大的集群、吸納最混亂的數據、產出最帶刺的回答。這不叫創新，這叫資源浪費。

如果 Grok 3 出來後還是那種「老子天下第一」的德性，卻連基本的邏輯推演都會出錯，那我建議馬斯克還是把那些 H100 拿去挖礦算了，至少那樣產出的價值還比較直接。現在這種半吊子的「真相機器」，除了給那些陰謀論者提供彈藥，真的看不出對文明有什麼貢獻。別再跟我提什麼 AGI 了，連個天氣預報和即時新聞都能總結得亂七八糟的模型，談 AGI 簡直是對這三個字的羞辱。

我看著那些數據中心在沙漠裡拔地而起，冷卻系統發出的低鳴聲彷彿是在嘲笑人類的焦慮。我們投入了這麼多資源，難道只是為了得到一個會發梗圖、會諷刺政客的聊天軟體？如果這就是人工智慧的巔峰，那人類的智慧也未免太廉價了。xAI 最好能拿出點真本事來，別再整天發那些虛無縹緲的基準測試截圖。數據可以造假，用戶的直覺不會。當一個產品好用到不需要老闆親自出來喊單時，那才叫真正的成功。

現在的現狀是，我們被迫在幾個各有缺陷的巨人之間做選擇。一個是道貌岸然的老師，一個是暮氣沉沉的學者，一個是經常斷片的富二代，還有一個是滿嘴跑火車的憤青。這就是目前 AI 領域的最高水平。你們可以在那邊吹捧 Colossus 有多宏偉，但在我看來，那只是一個巨大的、昂貴的、充滿電子噪音的荒原。在這片荒原上，還沒長出一朵真正有價值的花。

別再浪費時間去分析那些公關稿裡的技術細節了。那些指標不過是為了安撫投資人的止痛藥。真正決定勝負的，是誰能先擺脫那種「我教你做事」的優越感，誰能真正靜下心來處理數據中的垃圾，而不是把垃圾包裝成真相。Grok 還有機會，但這個機會不是靠算力贏來的，而是靠那些被他們嗤之以鼻的、細膩的、甚至是有點無聊的人文修補。如果他們還是打算一條路走到黑，那這場算力競賽最後只會留下一堆電子廢料和幾封過期的推文。
