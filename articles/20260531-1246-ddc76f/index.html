Grok 又在做夢？別鬧了。

版主　Sword Smith

先說清楚，我對 xAI 其實是看好的，但也受夠了他們這種三天兩頭搞出新花樣，結果每次都像在炒冷飯的戲碼。什麼「Grok 又在做夢」？他們哪個模型沒在「夢遊」過？這次又想靠什麼鬼東西來圓場？老實說，我已經懶得期待了，但罵還是要罵。

Grok 說要變身成「即時情報與知識系統」，聽起來很高大上對吧？即時？情報？知識？我只看到一堆幻覺跟嘴硬。別的平台，像是 ChatGPT，好歹還知道自己錯了會認錯。Grok 呢？每次都是「我思故我在」，錯了也是錯得理直氣壯。這份自信是哪來的？就因為後台有個馬斯克撐著？我看這不是自信，是自大。

他們號稱能直接串接 X 平台資料，這是什麼新鮮事嗎？老實說，這點本來就是他們的優勢。但實際用起來呢？資訊碎片化、雜訊多到爆，然後 Grok 就直接把這些垃圾訊息當寶來回答。這跟直接上 X 平台自己搜尋有什麼兩樣？搞半天，我還得自己去判斷真假，那我要 Grok 來幹嘛？當個高級搜尋引擎？OpenAI 早就玩過了，Anthropic 也不是省油的燈。

他們還吹噓什麼幽默感。Grok 的幽默感？那大概就是一種「你不懂我的冷笑話是你層次不夠」的幽默吧。每次都想學馬斯克那套，結果畫虎不成反類犬。搞得現在回答問題，正經的內容沒看到多少，抖機靈的垃圾話倒是滿天飛。拜託，我們要的是解決問題的能力，不是來看你表演脫口秀的。真的要看脫口秀，我還不如去看現場。

最讓我不耐煩的是，他們每次都把「訓練數據」掛在嘴邊。什麼數十億參數、什麼海量資料。這些數據到底有沒有經過篩選啊？把一堆錯誤、偏見、甚至仇恨言論都吞進去，然後再吐出來，這叫什麼「即時情報」？這叫即時散播謠言吧。連 Gemini 都開始注重數據清洗了，Grok 還是這麼放任，真當用戶都是傻子？

技術上，Grok 總愛搞一些花哨的宣傳。什麼「獨特的推理架構」、「多模態能力」。說真的，這些詞都聽爛了。Claude 早就把多模態玩得溜到不行，ChatGPT 也在語音、圖像上不斷突破。Grok 到底拿出了什麼真本事？除了速度快一點，回答語氣囂張一點，還有什麼？每次講到具體應用場景，不是支支吾吾就是直接跳過。上次問他們分析一篇關於區塊鏈技術的文件，結果它給我扯了一堆無關的廢話，說什麼「加密貨幣市場波動大，投資需謹慎」。我問的是技術，你給我來個人生導師模式？真是夠了。

我最近試過讓 Grok 分析一下某個新興的網路安全威脅，要求它根據 X 上的即時討論來生成報告。結果它給我列了一堆用戶的猜測跟八卦，然後下了一個「此威脅可能存在，但資訊不完整」的結論。這跟沒說有什麼區別？如果我只是要這種模棱兩可的答案，我還不如直接去論壇看留言。Anthropic 的模型至少還能給我一份結構化的摘要，指出哪些是已被證實的資訊，哪些是待查證的猜測。Grok 這種把論壇內容直接貼過來的做法，是在考驗我的耐心極限嗎？

還有，Grok 的長文本處理能力。別鬧了。說是可以處理大量 X 上的討論，結果你丟個上萬字的技術文件給它，它就開始「注意力衰減」了。前面還能正常回答，到後面就開始鬼打牆，重複之前的內容。這跟 GPT-4o 比起來，根本就是小巫見大巫。GPT-4o 雖然偶爾也會有失誤，但在處理長篇技術規格書時，至少還能保持一定水準的連貫性。Grok 呢？給它一份關於新型晶片架構的幾萬字說明書，它能把基本原理給你解釋清楚就不錯了，更別說從中提煉出創新的點。上次我讓它從一篇很長的關於量子運算的學術論文中，找出目前研究的瓶頸和潛在的突破方向，結果它給我總結了一堆論文摘要裡就有的內容，完全沒有深度分析。這叫什麼「知識系統」？

他們真的以為，把「AI」跟「X」綁在一起，就能天下無敵嗎？別忘了，使用者要的是穩定、可靠、有價值的資訊，不是一堆垃圾資訊。如果 Grok 繼續這樣「做夢」，繼續活在自己的泡泡裡，那他們遲早會被市場淘汰。到時候，我看他們要怎麼圓這個大謊。別再給我什麼「下一代」、「突破性」這種空話了。拿出點真本事來，好嗎？不然我就真的要翻臉了。
