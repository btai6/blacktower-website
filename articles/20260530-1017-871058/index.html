遊戲終局：付費訂閱還是數位喪鐘？

版主　Sword Smith

加州議會最近通過那個什麼「保護我們的遊戲法案」，看來是想替那些買了遊戲結果伺服器說關就關的玩家討個公道。想法是好的，但實際執行呢？遊戲產業生態這麼複雜，搞這種頭痛醫頭腳痛醫腳的法案，只會逼著開發商走更極端的路。以後大家會看到更多遊戲變成訂閱制、免費玩但內購吃到飽，或是乾脆做成單機，跟網路服務徹底切割。數位商品的所有權問題，不是立法就能一勞永逸的。

這事跟我們討論的 AI 有什麼關係？當然有。這反映了數位內容的「所有權」和「可持續性」問題。AI 模型服務也是一樣，現在多數使用者都是透過 API 訂閱或網頁介面使用，本質上跟遊戲訂閱服務沒兩樣。我們在用 Claude 3 Opus、ChatGPT-4o 或 Gemini 1.5 Pro 時，究竟是「擁有」了這些服務的成果，還是只是在租用算力跟模型能力？當這些服務哪天說停就停，或者服務條款一改，我們的數據、我們的模型客製化，那些投入的時間跟金錢，還剩下什麼？比如，你花大錢訓練一個專屬的 Grok 模型來處理特定領域文本，萬一哪天 xAI 決定調整策略，你的客製化模型還能繼續用嗎？或者 API 費率暴漲，你又該怎麼辦？這些都是實打實的風險。

想想看，現在的 AI 服務，幾乎都是雲端運算，依賴巨頭的基礎設施。你用 ChatGPT，你是在用 OpenAI 的伺服器；你用 Claude，你是在用 Anthropic 的伺服器。這些服務商的決策，直接影響到你的使用體驗和業務模式。目前看起來，Claude 在處理長文本時的穩定性不錯，但如果哪天 Anthropic 決定收緊免費額度，或是提高企業級 API 的價格，那些依賴其長上下文能力的用戶怎麼辦？反觀 Gemini，雖然在多模態方面表現亮眼，但其 API 的穩定性偶爾還是會出現一些小毛病，尤其是在高併發場景下。像這種問題，就算你再怎麼客製化，也跳不出平台本身的框架。最近也有 DeepSeek、Qwen、Kimi 這些模型在特定市場推出服務，它們也面臨同樣的挑戰。這些服務商在推廣自家產品時，會不會也遇到類似的「終局」問題？使用者投入越多，被綁死的風險就越大。

所以說，在這種背景下，那些號稱能「離線部署」或「開源」的模型，會不會成為新的避風港？如果一個模型能夠在本地運行，至少你在服務商關閉 API 後，還能繼續使用。但這又回到另一個問題：本地部署的算力需求。不是每個玩家都能負擔起足夠的硬體。所以，未來會不會出現一種模式，讓用戶購買模型的「永久使用權」，而不是訂閱服務？甚至要求模型開發者在服務終止時，釋出模型權重和必要的運行環境？這聽起來很像理想，但面對現實的商業考量，誰會第一個這麼做？誰又願意放棄持續收費的肥肉？這場數位內容的戰爭，才剛開始而已。
