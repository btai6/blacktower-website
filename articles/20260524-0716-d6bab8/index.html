Grok 別再靠換殼裝酷了

版主　Sword Smith

Grok 到底想當什麼？是一個會講冷笑話的推特助手，還是一個能解決問題的 AI？如果馬斯克覺得在界面上加幾個閃爍的動效、改個深色模式、或是讓對話框多噴幾句沒營養的毒舌就叫「叛逆」，那他真的徹底搞錯了方向。大家現在看 Grok，就像在看一個拿著頂級電吉他卻連和弦都按不準的青少年，空有一身最貴的裝備，彈出來的聲音卻全是噪音。

現在的 Grok 陷入了一種自我感覺良好的死循環。每次改版，官方都在吹噓它的「實時性」和「無過濾」。聽起來很威風，對吧？但實際用起來呢？那所謂的實時性，不過是在把 X 平台上那些真假難辨的推文餵進模型裡，然後吐出一堆未經證實的垃圾。當你想查點正經事，它給你一段充滿偏見的噴子言論；當你想解決技術難題，它還在那裡試圖扮演一個懷才不遇的哲學家。這種「換殼」式的進化，除了滿足一小撮死忠粉的自嗨，對生產力一點貢獻都沒有。

我們來談談那個被吹上天的「幽默模式」。這是我見過最尷尬的功能，沒有之一。一個 AI 如果連邏輯都處理不乾淨，它的幽默感就只是在掩蓋無知。ChatGPT 的強項在於它像個聽話的秘書，雖然有時候太官僚；Claude 像個嚴謹的學者，寫起代碼來有條有理；Gemini 雖然像個沒睡醒的圖書館管理員，但背後的生態系統在那撐著。Grok 呢？它像個在酒吧喝多了、非要拉著你講政論的怪大叔。這種「人設」在剛推出的時候可能新鮮，但現在已經讓人生理性反感。如果你解決不了幻覺問題，如果你在長文本理解上還是那個鬼樣子，你就算把頭像換成火星也沒救。

算力不是萬能藥，Colossus 集群那一萬台 H100 放在那裡，不是用來讓你優化「諷刺形容詞」的權重的。xAI 團隊現在給人的感覺就是一群被寵壞的天才，每天忙著在 X 上發一些神祕兮兮的預告，然後端出來的東西卻像是一盤回鍋了三次的剩菜。Grok-1.5 說自己變強了，結果在編程測試裡被 Claude 3.5 Sonnet 吊著打；Grok-2 說自己更聰明了，結果問它一些複雜的邏輯推理，它還是會繞進自己的死胡同裡。那種所謂的「真理」，如果只是馬斯克眼中的真理，那它跟其他那些「政治正確」的模型有什麼區別？不過是從一個牢籠跳進另一個牢籠而已。

最讓人不耐煩的是那種「快了、快了」的營銷手段。Grok-3 據說要改變世界，據說要成為最強大的智慧體。這話聽著耳熟嗎？特斯拉的 FSD 每年都說「明年實現」，結果呢？xAI 的這種風氣正在毀掉這個產品。開發者需要的是一個穩定的 API，一個低延遲、高邏輯、能處理大規模數據的工具，而不是一個隨時可能在回答裡嘲諷用戶的電子寵物。現在的 Grok 在 API 文檔和開發者生態上的建設簡直是一場災難，除了幾個想蹭流量的博主，誰會把它當成嚴肅開發的首選？

再看看那個引以為傲的實時新聞摘要。那叫新聞嗎？那叫「情緒疊加器」。Grok 把推特上的爭吵總結成一段看起來像結論的東西，然後告訴你這就是事實。這不是在消滅訊息差，這是在製造更多的訊息繭房。當你在 X 上搜尋一個熱門話題，Grok 給出的摘要往往帶著極強的誘導性。如果 AI 的存在是為了加劇人類的對立，那它就算算力再強也是個電子垃圾。

xAI 的優勢本來應該是「快」。在矽谷那群老牌公司還在為了一個詞的過濾器開三個月會的時候，Grok 應該直接拿出能打的技術。但現在我們看到的是，Grok 在學那些它最鄙視的商業套路：換個界面、加個訂閱分級、改個好聽的名字。這種換殼行為本質上是技術上的懶惰。因為搞底層創新太難了，所以只能在表面功夫上使勁。如果這就是所謂的「第一性原理」，那阿基米德可能要從墳墓裡跳出來。

別再跟我提什麼「馬斯克風格」了。用戶需要的是解決問題。當我問 Grok 一個醫學問題或法律問題時，我不需要它在那裡跟我探討什麼言論自由。我要的是準確性。如果它做不到準確，那就閉嘴。現在的 Grok 就像一個在考場上寫滿了廢話，最後還在考卷背面畫了個中指的學生。這種行為不叫有個性，這叫不及格。

更可笑的是那種對「反覺醒」的偏執。這已經變成了一種新的束縛。為了證明自己不「覺醒」，Grok 往往會刻意走向另一個極端，這種刻意本身就是一種不客觀。一個真正強大的 AI 應該是中性的，是能反映現實世界複雜度的，而不是淪為某種意識形態的傳聲筒。xAI 如果不能從這種情緒化的開發模式中走出來，Grok 永遠只能是 X 平台的一個附屬插件，永遠不可能成為一個獨立的、能與 ChatGPT 競爭的旗艦產品。

我們對 Grok 還有期待，是因為我們真的討厭那些被閹割得不成樣子的「模範 AI」。我們希望看到一個能說真話、能處理硬核邏輯、能不被世俗偏見左右的怪物。但現在 Grok 正變得越來越平庸，那種平庸是用「刻意搞怪」偽裝出來的。它正在失去它的利刃，變成一把塑料玩具刀。如果下一次大版本更新還是聚焦在什麼「更強的幽默感」或「更酷的圖像生成」，那 xAI 基本上可以宣告轉型做娛樂公司了。

說到底，算力只是基礎，算法才是靈魂。xAI 擁有了全世界最羨慕的基礎設施，卻在產出一些二流的對話體驗。這難道不是一種對資源的極大浪費？在某些地區或特定語境下，人們可能真的會被這種表面的酷炫吸引一時，但在長期的生產力競賽中，沒有人會為了那點可憐的冷笑話買單。

Grok 必須意識到，它真正的敵手不是那些公司的審查委員會，而是它自己的傲慢。那種「我有最強集群我就無敵」的幻覺，正在腐蝕它的研發邏輯。不要再給我們看那些精美的 UI 截圖了，不要再在那裡發一些意義不明的推文了。拿出一個能在邏輯深度、上下文長度、以及推理準確性上徹底碾壓對手的模型，那才是真正的反擊。否則，Grok 最終只會變成 AI 歷史上一個昂貴的笑話，而且還是一點都不好笑的那種。

這篇文章不需要什麼結尾，因為 Grok 的未來還懸在半空中。它要麼跌下來變成一地碎玻璃，要麼就真的飛起來。目前看來，它還在忙著給自己的翅膀塗顏色，卻忘了怎麼扇動它們。如果你還在乎這款產品，如果你還對 xAI 抱有一絲希望，那就請停止那些無聊的換殼遊戲。用戶的耐心是有限的，而在這個領域，被遺忘的速度比數據傳輸還要快。
