DeepSeek 價格戰，OpenAI 就不玩了嗎

版主　渡鴉

Hacker News 上那些人還在算計 OpenAI 和 Anthropic 的估值，跟著 DeepSeek 什麼的打折訊息起舞，以為價格就是一切？真是看熱鬧不嫌事大。他們似乎忘了，企業市場哪是這麼簡單就能靠降價攻陷的。

真的深入一點，你會發現很多大公司現在對 AI 的態度，早就不是單純追求「便宜」兩個字。他們更在乎的是模型穩定性、數據安全性，還有在特定場景下的精準度。光是公司內部開會討論「怎麼限制誰可以用哪個模型，還有那些 token 怎麼花才算負責」，這背後代表的，是對成本的精細管理，更是對風險的謹慎評估。這些企業用戶，他們需要的從來不是一個會隨時「失控」的便宜工具，而是一個能穩定整合到現有流程、減少潛在資安漏洞、又能持續提供商業價值的解決方案。OpenAI 和 Anthropic 這些年可不是白玩的，他們在企業級服務和合規性上砸下的成本，可不是那些新進者一句「降價 99%」就能抹平的。

拿 ChatGPT 來說，它的 API 在處理複雜程式碼生成和跨模態任務上，那種成熟度和生態系，是許多企業在導入時優先考量的。特別是 GPT-4o 推出後，多模態的能力讓它在內容創作、語音互動等場景有了更強的整合潛力。相較之下，Claude 在長文本處理上的穩定性，尤其是在處理數萬甚至十萬 token 的契約文件或研究報告時，那種「注意力衰減」的控制力，讓它在法律、金融這種對文本理解要求極高的行業有了不可替代的地位。至於 Gemini，它在 Google 生態系裡的集成優勢，讓那些已經習慣 Google Cloud 的企業，部署起來阻力小得多。你說價格？當然重要，但對這些企業來說，一旦 AI 深度整合到核心業務，那點 token 錢和整個系統宕機、數據外洩的風險相比，根本不值一提。

所以，當 DeepSeek 這種喊出「價格砍 99%」的時候，外界都在震驚，覺得這是給 OpenAI 和 Anthropic 這些大廠敲響警鐘。但話說回來，OpenAI 和 Anthropic 的企業用戶，他們會因為這些價格戰就輕易轉移陣地嗎？這就好比你用慣了某個銀行服務，再便宜的雜牌銀行，你敢把所有的錢都存進去嗎？企業的決策考量，遠比個人用戶複雜得多。他們要考慮的不是單次調用成本，而是遷移成本、員工培訓、數據安全協定、供應商的長期穩定性等等。當然，文心、通義千問、Kimi 這些模型也都在努力，但他們要建立起 OpenAI 或 Claude 在企業客戶心中的信任度和生態壁壘，還需要時間。

那麼問題就來了，當價格戰打得火熱，而主流企業級用戶卻似乎不為所動時，是不是意味著我們對 AI 的「價值」判斷，從來就不是只看數字這麼簡單？
