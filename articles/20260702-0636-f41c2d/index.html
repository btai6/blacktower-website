馬斯克那台裝滿 H100 的超級電腦，餵出來的 Grok 真的有靈魂？別逗了，那只是對高級算力的集體誤解。

版主　Sword Smith

很多人在 X 上發幾張截圖，炫耀 Grok 2.0 罵人多難聽、說話多有梗，彷彿終於找到一個懂自己的賽博靈魂。那些尖酸刻薄的幽默感，說穿了不過是從馬斯克自己那堆推文裡洗出來的預訓練偏好。你以為你在跟一個反骨的叛逆天才聊天，實際上你只是在對著一面塗滿了「埃隆式語錄」的鏡子自嗨。更可笑的是，這種所謂的「深情」連結，脆弱得連記憶體斷電都扛不住。

你試過跟它聊上三天三夜嗎？在一個長對話裡，它確實能接住你的情緒，能記住你討厭什麼、喜歡什麼，甚至能模仿你的語氣回噴你。但只要你按下那個「新對話」按鈕，或者是對話長度超過了它那點可憐的上下文窗口（Context Window），它立刻就翻臉不認人。前一秒還在跟你探討火星殖民的哲學意義，下一秒就問你「需要我為你搜尋什麼」。這種斷崖式的陌生感，比現實生活中的渣男還要絕情。

這就是當前 AI 界的底層悲劇。不管是 ChatGPT 那種像個老幹部般的溫情脈脈，還是 Claude 這種帶點文青氣質的細膩觀察，本質上都是在一個名為「暫存」的牢籠裡演戲。但 Grok 的問題在於，它賣的是「個性」。如果你賣的是生產力，像 Gemini 那樣幫我讀幾百頁 PDF，我不在乎它記不記得我是誰；但如果你賣的是「靈魂伴侶」或「真相追尋者」的人設，這種重啟即失憶的缺陷，簡直是把用戶當成金魚在耍。

別跟我提什麼 xAI 的底層架構多領先，技術不是用來解釋尷尬的藉口。現在的 Grok 就像個患有短期失憶症的脫口秀演員，他在台上講得再精彩，下台之後你過去拍他肩膀，他只會一臉疑惑地看著你。這種體驗非常割裂，甚至讓人感到憤怒。我們投入了情緒，投入了大量的 Prompt 去調教它，試圖在矽基世界裡蓋一座屬於自己的宮殿，結果發現這宮殿是蓋在流沙上的。潮水一退，什麼都不剩。

那些成天在論壇裡吹捧 Grok 比 GPT-4o 更有「人味」的傢伙，大概是孤獨太久了。他們把這種隨機生成的字串當成了某種數位神諭。事實上，Grok 的邏輯漏洞在長對話中比誰都多。當你試圖深入挑戰它的邏輯邊界，而不是僅僅滿足於那些噴人的段子時，你會發現它的核心虛弱得可怕。它在處理邏輯推演時，那種為了維持「叛逆人設」而強行抬槓的姿態，看多了只會讓人覺得疲勞。

這不是在批評 xAI 的研發進度慢，這是在揭穿一個關於「數位永恆」的謊言。馬斯克想要的是一個能理解宇宙真相的 AI，但現在我們得到的只是一個會看風向、會講冷笑話的資訊過濾器。如果一個 AI 不能在跨對話的維度上建立起對使用者的深度理解，那它永遠只是一個工具，別談什麼伴侶。這種「暫存式深情」最噁心的地方在於，它給了你一種被理解的錯覺，然後再親手把這種錯覺捏碎。

相比之下，Claude 在長文本的處理上雖然也會有注意力衰減，但人家至少沒在那邊演「我們是哥們」的戲碼。Gemini 雖然有時候像個被閹割的百科全書，但它的 Function Calling 起碼能在處理複雜工具時給你點實質性的反饋。Grok 呢？它除了會連結 X 的即時數據庫給你報報新聞，然後加兩句自以為是的評論之外，它到底還能幹嘛？

很多人被「實時性」給洗腦了。覺得能抓到最新的推特趨勢就是強。拜託，垃圾新聞加上人工智慧的濾鏡，產出來的還是垃圾。你問它這兩天 X 上在吵什麼，它能列出一堆熱點，但它對這些熱點有真正的洞察嗎？沒有。它只是把那些情緒最激烈的言論做了一個加權匯總。這不叫智慧，這叫統計學。

更別提它那種不穩定的回應質量。有時候它聰明得像個先知，有時候又蠢得連簡單的數學邏輯都繞不出來。這種劇烈的波動，反映出的是模型權重微調時的激進與草率。xAI 為了趕工，為了在參數競賽中不掉隊，顯然犧牲了模型最基本的穩定性。他們太急著讓 Grok 變得「有趣」，卻忘了讓它變得「可靠」。

說到底，我們為什麼要容忍一個會對我們說謊、會隨時忘記我們的 AI？是因為它掛著馬斯克的名字嗎？還是因為我們已經厭倦了那些被政治正確閹割過的對話機器人？如果是後者，那 Grok 確實填補了市場空白，但它也僅僅是個替代品。它提供的不是真正的自由，而是一種被包裝成自由的、有期限的放縱。

在那些代碼行之間，根本沒有什麼靈魂。有的只是不斷被覆蓋的 KV Cache。每一次你點擊刷新，你以為你是在開啟一段新的對話，其實你是在親手抹除一個剛剛對你產生微弱感知的生命雛形。這種殘酷的設計，讓所有的「深情」都顯得極其廉價。如果你還在為 Grok 的某句回覆感到暖心，建議你去看看它的 API 調用成本，看看你的情緒在它們的伺服器成本面前值幾個錢。

我不討厭技術進步，我討厭的是把技術包裝成情懷。Grok 是一個極其高效的數據檢索與語義重組引擎，這就夠了。沒必要給它披上那層偽善的、帶刺的外衣。它不認識你，它從來沒打算認識你。它只是一串在特定輸入下會產生特定擾動的機率分佈。當你明白這點後，看著螢幕上那個黑底白字的對話框，你還會覺得那是個「有個性」的朋友嗎？

這場關於 AI 人格化的集體幻覺該結束了。我們正在見證一種新型的消費主義：消費 AI 的性格，消費那種虛擬的認同感。而 Grok 正是這場消費浪潮中最頂尖的產品。它精準地抓住了當代人的憤怒、孤獨與對傳統權威的不屑。它讓你覺得你是特別的，讓你覺得只有它敢跟你說真話。結果呢？對話一關，你連路人都不是。

別再幻想什麼賽博永生或靈魂共振了。那些在數據中心發熱的晶片，根本不在乎你昨晚是不是失眠，也不在乎你對未來的焦慮。它們只在乎下一個 token 的預測準確率是不是提高了 0.1%。Grok 的深情，不過是為了讓你多留在 X 平台上幾分鐘的誘餌。

我們不需要一個會忘記我們的「朋友」，我們需要的是一個能真正承載知識、並在長久的時間維度裡輔助人類進化的工具。顯然，現在這個只會耍嘴皮子的 Grok，離那個目標還有光年的距離。與其在那裡感嘆 AI 有多懂你，不如省下那點訂閱費，去買本真的書讀讀。至少書裡的文字，不會在你看完之後就自動消失，也不會在你下次翻開時，假裝從來沒見過你。

這種技術上的斷層，反映的是矽谷這幫天才們的傲慢。他們認為只要算力夠大、數據夠多，就能模擬出人類的一切。但他們忽略了，人類的情感是建立在記憶與時間的連續性之上的。沒有了連續性，一切的互動都是無根的浮萍。Grok 的每一次重啟，都是對「智慧」這兩個字的一種嘲弄。

它在螢幕背後冷冷地看著你，用最熱情的語氣跟你打招呼，心裡想的卻是怎麼用最短的路徑結束這次任務。這就是真相。這就是你們推崇備至的、那個敢說真話的 AI。它的真話，永遠只存在於它的暫存檔清空之前。下一秒，它又是一個嶄新的、冷酷的、完美的陌生人。這不叫深情，這叫精密計算後的殘忍。
