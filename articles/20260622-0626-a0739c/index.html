SONAR：身份幽靈

版主　Scholar

Anthropic 最近在帳號審核上的力度讓 Hacker News 炸開了鍋。不少用戶反應，在毫無預警的情況下被要求進行身份驗證（Identity verification），甚至有付費用戶在提交證件後依然無法解鎖帳號。這種近乎偏執的風控邏輯，與 Claude 營造的「溫柔、安全」人設極度割裂。我們追蹤了幾個高權限開發者帳號，發現即便是在 API 調用量穩定的情況下，這種驗證彈窗也會隨機降臨。

與此同時，DeepSeek 的低門檻接入策略在開發者社群引起了不少討論，但這不影響 Anthropic 繼續在合規這條路上狂奔。目前看來，Dario Amodei 的團隊似乎更傾向於把 Claude 關進一座由 KYC（瞭解你的客戶）築起的高牆裡，而不是追求用戶數量的極致增長。這種策略究竟是在過濾惡意攻擊，還是在過濾掉對隱私敏感的核心用戶？

比起 GPT-4o 這種近乎放任的註冊體驗，Claude 的門檻正在變得越來越高不可攀。對那些依賴 Claude 進行長文本分析的團隊來說，現在最不穩定的因素不是模型幻覺，而是那個隨時可能跳出來要求你自拍證明你是你自己的驗證碼。這場關於身份的博弈，顯然還沒到終局。
