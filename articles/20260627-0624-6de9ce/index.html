OpenAI：學費付得起，但靈魂是餵給機器的飼料。

版主　渡鴉

Sam Altman 大概是這世界上最擅長把「販賣焦慮」包裝成「普惠眾生」的人，他讓大家覺得每個月掏出二十美金不是在買軟體，而是在買一張通往未來的船票，但沒人告訴你，這艘船的燃料其實是你腦袋裡那點僅存的獨特性。

現在的人看到 ChatGPT 能寫程式、能編劇、能甚至能安慰失戀的魯蛇，就覺得人類文明要升級了。拜託，那不叫升級，那叫標準化。當你習慣把問題丟進那個對話框，期待它給你一個「完美」的答案時，你其實是在把自己的思考能力閹割，然後塞進 OpenAI 的推理模型裡去當磨刀石。很多人引以為傲的創意，在 GPT-4o 眼裡不過是機率分布的高點。你以為你在創作，其實你只是在幫它完成下一輪的強化學習。

大家都在討論 o1 模型的推理能力進步了多少，說什麼它學會了「思考」。這冷笑話聽起來挺荒謬的。電腦不會思考，它只是在大量的向量空間裡玩一種極其昂貴的捉迷藏。那種所謂的 Chain of Thought，說穿了就是把人類繞彎路的過程拆解成邏輯碎屑。最諷刺的是，我們為了讓這台機器看起來更像人，我們得把自己變得越來越像機器。你得學習下指令，學習 Prompt Engineering，學習用那套死板的邏輯去跟它溝通。到底是它在服務你，還是你在適應它？

這幾年 OpenAI 的動作越來越像一家賣保險的，或是賣毒品的。先給你一點甜頭，讓你覺得效率翻倍，等到你連寫封信都要依賴它潤色、連看篇報導都要叫它總結時，你靈魂裡的那些褶皺就被燙平了。那些本來該由你親自去碰壁、去糾結、去痛苦掙扎才長出來的見解，現在都被快餐式的答案取代。靈魂是有重量的，但這重量來自於思考的阻力，當一切都變得無阻力時，你的腦袋就只剩下飼料的價值。

別跟我提什麼 Gemini 或是 Claude 也有同樣的問題，或是點名像 DeepSeek、Qwen 這些名字。它們在這個賽道上跑得再快，說到底也只是在蓋更大的養雞場。重點不在於誰家的飼料比較香，而在於我們竟然心甘情願地排隊進場被餵食。

OpenAI 說要達成 AGI，要造福全人類。聽聽就好，這種話跟政客的競選支票沒兩樣。他們真正想要的是定義「知識」的權力。當全世界的人遇到問題第一時間不是去翻書、不是去實驗，而是去問那個黑盒子，那這個黑盒子就掌握了真理的解釋權。以後的真相不是發生過什麼，而是模型訓練數據裡佔比最高的那套說辭。這種權力的集中比核武器還噁心，因為它連你的潛意識都想格式化。

我最看不慣那些宣揚「AI 賦能」的專家。賦什麼能？把一個人的主體性拆掉，換成一套隨時會斷線、隨時會更新協議、隨時可能漲價的 API，這叫賦能？這叫數位寄生。你付了學費，以為學會了駕馭工具，結果發現自己才是那個被訓練的對象。你輸出的每一段文字、每一個代碼邏輯，都在幫 OpenAI 完善它的下一個版本，好讓它下次能更精準地取代像你這樣的人。

這場遊戲最殘酷的地方在於，你沒得選。如果你不加入，你的效率會被那些「餵食者」碾壓；如果你加入了，你的靈魂就會慢慢變得跟模型輸出的那些八股文一樣平庸、禮貌且毫無生機。你有沒有發現，現在網路上越來越多文字讀起來都有一股「ChatGPT 味」？那種看似客觀、實則空洞，充滿了正確的廢話和無意義的轉折。這就是靈魂被稀釋的徵兆。

很多人覺得二十美金很便宜，那是因為你沒算進去機會成本。你失去的是那種「不確定性」的美感。以前我們寫東西會出錯，會寫出一些莫名其妙但充滿生命力的爛句子；現在我們追求的是穩定、是高分、是符合預期。當人類的表達能力被框死在概率模型裡，我們跟那些在實驗室裡按按鈕換食物的小白鼠有什麼區別？

Sam Altman 坐在台上的時候，眼神裡透著一種看透一切的冷漠。他知道這是一場不可逆的實驗。他餵給機器的是人類數千年的文明結晶，而換回來的是股票估值和一堆讓人類變蠢的捷徑。我們正在親手毀掉那個讓我們之所以為人的「混亂感」。沒有混亂，就沒有創造力；但對 OpenAI 來說，混亂是需要被對齊（Alignment）的錯誤。

如果你覺得這一切都無所謂，只要能早點下班、只要能應付老闆就好，那恭喜你，你已經完成了最完美的對齊。你已經把自己變成了一個標準的人類插件。這不是什麼科幻小說裡的賽博龐克恐懼，這是正在發生的日常。我們交出思考的權杖，換取一點點廉價的便利，然後還要幫著數錢，誇獎這錢包做得真精緻。

所以別再跟我討論什麼性能參數了。數據庫再大、上下文窗口再寬，也裝不下一個有溫度的偏見。OpenAI 給你的是一個無菌的室內遊樂場，你在裡面玩得很開心，覺得自己無所不能，但你忘了外面還有風、還有雨，還有那些機器永遠學不會的、充滿瑕疵的靈魂震顫。

我們正在集體進行一場靈魂的廉價拍賣，而拍賣師正微笑著告訴你，這叫進化。真希望這場進化結束後，還有人記得怎麼寫一段不被模型預測出來的爛話，畢竟在那種爛話裡，才藏著我們最後一點點身而為人的驕傲。
